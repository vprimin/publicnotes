          iTerm2

          Created on May 30, 2024, 1 min read

          Windows>Hide scrollbars

          Текст Ставим Шрифт крутой Source+Code+Pro Называется Profiles>Colors>Text или JetBrainsMono Тема: Solarized Dark Window: Transparency - 8, Blur - 15

          Profiles>Colors>Color Presets>Solarized Dark (for dark mode)

          oh-my-zsh + Plugins

          Показать что можно сохранять профили в json Save profile into .json

          Показать как установить тему из галлереи https://iterm2colorschemes.com/ - добавим новую цветовую схему, сохраним с расширением .itermcolors Violet Light Например

          Иконка, показать как поменять иконку

          git clone git@github.com:jasonlong/iterm2-icons.git ~/iterm2-icons

          1. Go to Applications folder, select iTerm and hit Cmd+I (or right-click -> Get Info)

          2. Open the ~/iterm2-icons folder.

          3. Drag the .icns icon of your choice onto the old iTerm2 icon.
