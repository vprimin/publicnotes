
В Микро есть поддержка мыши, темы, ==подсветка синтаксиса==, плaгины.

>Минус в том, что микро не входит в стандартный набор большинства дистрибутивов

site: https://micro-editor.github.io/


```
curl https://getmic.ro | bash

sudo mv micro /usr/local/bin 
```

Installation:  `brew install micro`

```
micro ~/.zshrc
```
